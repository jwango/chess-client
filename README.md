# Garden Sketch

## Setup

### Installation
1. Install Node 18 (recommend managing versions via a manager like `asdf` or `nvm`)
2. `npm install --global yarn`
3. `yarn install`

### Development
* Develop via `yarn start`
* Build for production via `yarn build` - make sure `NODE_ENV=production`
